# 💖 Hair Care Advisor 💆‍♀️✨

A tiny self-care inspired program that helps you understand your hair better and choose the right oils for it 🌸

---

## 🌷 About This Project

This little project was created during a moment of *“I should really start taking better care of myself”* 💭💖

Instead of just thinking about it, the I decided to build something cute and helpful — not just for myself, but for **everyone who wants to understand and love their hair more** ✨

Because self-care isn’t selfish… it’s necessary 💅💕

---

## 💡 What It Does

This program:

* 🧠 Asks simple questions about your hair
* 💧 Determines your **hair porosity** (Low, Medium, High)
* 💆‍♀️ Suggests the best **hair oils** for your hair type

---

## 🌸 Features

* 💖 Beginner-friendly Java code
* 🫧 Quick and easy hair quiz
* 🌿 Personalized oil recommendations
* ✨ Cute + self-care vibes

---

## 🛠️ How It Works

1. Answer a few questions about your hair
2. The program calculates your **hair type**
3. You get oil suggestions tailored just for you 💕

---

## ▶️ How to Run

1. Save the file as `HairCareAdvisor.java`
2. Compile:

   ```
   javac HairCareAdvisor.java
   ```
3. Run:

   ```
   java HairCareAdvisor
   ```

How long does your hair take to dry?
